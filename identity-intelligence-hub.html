CORS Proxy (opzionale ma consigliato)

La maggior parte dei provider OSINT (LeakIX, BreachDirectory, IntelX, DeHashed) non consente chiamate dirette dal browser per via delle policy CORS. Configura un proxy per inoltrare le richieste. Esempi pubblici: https://corsproxy.io/?, https://api.allorigins.win/raw?url=. Per uso reale, deploya un proxy tuo (Cloudflare Worker, Nginx, etc.).

Limite proxy pubblici: servizi come corsproxy.io spesso strippano gli header custom (x-key, Authorization, X-RapidAPI-Key) — quindi le API che usano questi header per autenticarsi falliscono comunque. Per usare davvero questa app deploya il tuo Cloudflare Worker (vedi sotto, 2 minuti gratis).

Anteprima URL chiamata: —
Esempi accettati: https://tuo-worker.workers.dev/ · https://tuo-worker.workers.dev/? · https://corsproxy.io/?. Il ? finale viene aggiunto automaticamente.

Deploy del tuo Cloudflare Worker (consigliato)

  1. Vai su dash.cloudflare.com → Workers & Pages → Create → Hello World.
  2. Copia il codice qui sotto e incollalo nell'editor del Worker, poi Deploy.
  3. Copia la URL che Cloudflare ti dà (es. https://iihub-proxy.tuo-account.workers.dev/?) e incollala nel campo proxy qui sopra.
const CORS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Methods': 'GET, POST, PUT, DELETE, OPTIONS',
  'Access-Control-Allow-Headers': '*',
  'Access-Control-Expose-Headers': '*',
  'Access-Control-Max-Age': '86400'
};

export default {
  async fetch(request) {
    // Preflight CORS: rispondi subito senza chiamare l'upstream
    if (request.method === 'OPTIONS') {
      return new Response(null, { status: 204, headers: CORS });
    }

    const url = new URL(request.url);
    const target = url.search.slice(1); // tutto dopo il "?"
    if (!target) {
      return new Response('Missing target URL. Usage: ' + url.origin + '/?https://api.example.com/path',
        { status: 400, headers: { ...CORS, 'Content-Type': 'text/plain' } });
    }

    let targetUrl;
    try { targetUrl = new URL(decodeURIComponent(target)); }
    catch { return new Response('Invalid target URL', { status: 400, headers: CORS }); }

    // Inoltra header del client, rimuovendo quelli che non vanno propagati
    const headers = new Headers(request.headers);
    ['host','origin','referer','cf-connecting-ip','cf-ipcountry','cf-ray','cf-visitor','x-forwarded-for','x-forwarded-proto','x-real-ip']
      .forEach(h => headers.delete(h));
    // Alcuni provider rifiutano richieste senza User-Agent
    if (!headers.has('user-agent')) headers.set('user-agent', 'IIHub-Proxy/1.0');

    let upstream;
    try {
      upstream = await fetch(targetUrl.toString(), {
        method: request.method,
        headers,
        body: ['GET','HEAD'].includes(request.method) ? undefined : request.body,
        redirect: 'follow'
      });
    } catch (e) {
      return new Response('Upstream fetch failed: ' + e.message,
        { status: 502, headers: { ...CORS, 'Content-Type': 'text/plain' } });
    }

    const respHeaders = new Headers(upstream.headers);
    Object.entries(CORS).forEach(([k, v]) => respHeaders.set(k, v));
    return new Response(upstream.body, { status: upstream.status, headers: respHeaders });
  }
};
